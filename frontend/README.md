# 🌐 Hackitise Labs - Frontend Platform

A premium, production-grade cybersecurity marketing platform built with **React 19** and **Vite**.

## 🎨 Design Philosophy
- **Cyber-Aesthetic**: Modern dark themes with neon accents, glassmorphism, and interactive grid backgrounds.
- **Motion-First**: Smooth transitions and scroll-triggered animations powered by **Framer Motion**.
- **Responsive**: Mobile-first design ensuring a seamless experience across all devices.

## 🚀 Development

### Scripts
- `npm run dev`: Launch Vite development server.
- `npm run build`: Generate production bundle.
- `npm run lint`: Run ESLint for code quality checks.

## 🏗️ Architecture

### Component Organization
We use a structured component system for maximum reusability:

1. **`layout/`**: Global components like the **Navbar** and **Footer**.
2. **`sections/`**: Modular page sections (e.g., **Hero**, **Testimonials**, **Pricing**).
3. **`ui/`**: Reusable base components (e.g., **Logo**, **AnimationUtils**, **Carousel**).

### Styling
- **Tailwind CSS 4**: Utilizing the latest features for utility-first styling.
- **Custom Theme**: Defined in `tailwind.config.js` with semantic color tokens.

## 🛠️ Key Libraries
- **Framer Motion**: Advanced animations.
- **Lucide React**: Clean and consistent icons.
- **React Router 7**: Client-side routing.

## 🧭 Pages
- **Home**: Dynamic overview of services and trust signals.
- **About**: Company mission, mentors, and leadership team.
- **Services**: Detailed breakdown of VAPT, Compliance, and Training.
- **Contact**: Interactive lead generation form with map integration.
