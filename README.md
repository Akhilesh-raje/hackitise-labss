# 🛡️ Hackitise Labs - Enterprise Cybersecurity Monorepo

Welcome to the **Hackitise Labs** official platform. This repository houses the complete ecosystem for our cybersecurity firm, including a high-performance marketing frontend and a robust TypeScript-based API service.

## 📁 Repository Structure

This monorepo uses **npm workspaces** to manage its components:

- [frontend/](file:///c:/Users/rajea/Documents/hakesite%20-%20Copy/frontend/) - Vite + React 19 platform with Framer Motion animations.
- [backend/](file:///c:/Users/rajea/Documents/hakesite%20-%20Copy/backend/) - Express 5 + TypeScript API service.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- npm (v9+)

### Installation
From the root directory, run:
```bash
npm install
```

### Development
Launch both frontend and backend concurrently:
```bash
npm run dev
```

- **Frontend**: [http://localhost:5173](http://localhost:5173)
- **Backend**: [http://localhost:4000](http://localhost:4000)

## 🛠️ Technology Stack

### Frontend
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS 4 (Custom Design System)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM 7

### Backend
- **Framework**: Express 5
- **Language**: TypeScript 6
- **Runtime**: tsx (watch mode)

## 📖 Component Architecture

The frontend follows a modular architecture:
- **`src/components/layout`**: Persistent UI like Navbar and Footer.
- **`src/components/sections`**: High-level page sections (Hero, Services, FAQ).
- **`src/components/ui`**: Atomic UI elements and utility functions.
